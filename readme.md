Um sistema inteligente e colaborativo para monitoramento de preços de combustíveis em **São José do Rio Preto - SP**. O projeto utiliza mapas interativos para conectar motoristas aos postos com os melhores valores da região.

---

## ✨ Funcionalidades

- 🗺️ **Mapa Interativo**: Visualização em tempo real de postos de combustível em SJRP usando **Leaflet.js** (alternativa gratuita ao Google Maps).
- 📊 **Consenso de Preços**: Sistema inteligente que calcula a confiabilidade do preço com base no tempo e número de relatos.
- 📍 **Geolocalização**: Acompanhamento da posição do usuário e cálculo de distância até os postos.
- 🧮 **Calculadora de Litros**: Utilitário integrado para saber exatamente quanto vai entrar no tanque antes de abastecer.
- 🏥 **Pontos de Interesse (POIs)**: Exibição de hospitais, fast-food e shoppings para auxiliar na navegação urbana.
- 📱 **Interface Mobile-First**: Design moderno e responsivo, pronto para ser usado como PWA (Web App).

## 🚀 Tecnologias Utilizadas

- **React.js**: Biblioteca principal para interface.
- **Leaflet**: Renderização de mapas de código aberto.
- **OpenStreetMap & CartoDB**: Provedores de camadas de mapa (Tiles).
- **Tailwind CSS**: Estilização rápida e responsiva.
- **Lucide React**: Biblioteca de ícones modernos.
- **Vite**: Ferramenta de build rápida.
